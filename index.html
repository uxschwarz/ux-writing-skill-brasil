Get started

Claude Desktop

Upload directly

In Claude Desktop, go to Settings → Capabilities → Upload skill, then select the downloaded skill file.

Claude Code

Manual install

Copy the ux-writing folder to ~/.claude/skills/ and restart Claude Code to activate.

Step three

Use

Ask Claude to review any interface copy and it will automatically apply UX writing best practices.

→ Review this empty state message
"You don't have any saved items."
 
✓ Analyzing against 4 quality standards
 
Evaluation:
✗ Purposeful: Doesn't help user take action or achieve goals
✓ Concise: Word count is appropriate (6 words)
✗ Conversational: Negative framing, sounds discouraging
✓ Clear: Message is unambiguous
 
Issues:
• Missing call-to-action to populate the empty state
• Negative framing ("don't have") instead of neutral/positive
• No guidance on what to do next
 
Recommended:
"No saved items yet. Browse products to start building your list."
 
Why this works:
• Purposeful: Clear CTA guides user to next action
• Conversational: "yet" implies this is temporary and expected
• Follows empty state pattern: Explanation + CTA
