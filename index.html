Figma integration

Review and improve UX copy directly from your Figma designs. Perfect for content designers and product teams.

Quick setup

Then share any Figma frame link with Claude to get instant UX writing feedback, accessibility audits, and specific improvements.

Get started

Step one

Download

Download the skill and extract the ZIP file to find the ux-writing folder.

Step two

Install

Copy the ux-writing folder to ~/.claude/skills/ and restart Claude Code.

Step three

Use

Ask Claude to review any interface copy and it will automatically apply UX writing best practices.
